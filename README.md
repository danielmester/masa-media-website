# מסה מדיה דיגיטל — דף בית חדש

דף בית חדש לאתר [masamedia.co.il](https://www.masamedia.co.il/) — חברת קידום אתרים אורגני (SEO) ואופטימיזציה למנועי בינה מלאכותית (GEO).

עיצוב בהיר בסגנון בוטיק, RTL מלא, עם גרדיאנט המותג (טורקיז→סגול) כאקסנט. כל התוכן — שירותים, חבילות, מקרי בוחן, עדויות, FAQ, תפריט ופוטר — נלקח מהאתר הקיים.

## הרצה מקומית
זהו אתר סטטי. פשוט פותחים את `index.html` בדפדפן, או:

```bash
npx serve .
```

## מבנה
- `index.html` — הדף המלא (HTML + CSS + JS בקובץ אחד)
- `assets/` — לוגו, favicon, ולוגואי לקוחות
